GamsJS Documentation

What is GamsJs?

A simple library trying to harmonize common imperative js patterns related to GAMS development, for example for building correct urls (for query objects) or copying text to the client’s clipboard.

This lib is not meant to be used with any js frameworks (like React, Vue, Angular, Svelte or else) as it focuses on support for server side handled application state and business logic. (Following a strict MVC approach).

Whats inside GamsJs?

  • copy text to clipboard (add when clicked on an element)
  • building correctly encoded GAMS Urls at client side
  • groupBy helper function
  • demand the client to switch to https
  • programmatically open given url in new tab
  • js redirect client (current )
  • helpers for correctly aligning fixed navigation bars

Recommended Setup

Setup (GAMS) Do not use a local copy of gams.js instead include it from a centralized and versioned location.The standard distiribution location is: "/lib/3.0/gamsJS/1.x/gams.js"You may copy the gams.js setup from the quickstart section above.
Versioning gams.js tries to only support one version inside the 1.x folder. There are no API breaking changes planned. gams.js tries to be as simple as possible.

GAMSUrl - Class

Extension of js built-in URL class by GAMS specific features.

Description Extends the js built in URL-class by GAMS specific features like "pid", "sdef", and functionality to build the params sub-query. Handles the required url-encoding to create propper query-urls.E.g. best used in conjunction with other gamsJs helper methods for sending client to created query-urls in new tab.
