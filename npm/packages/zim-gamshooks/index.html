ZIM GAMS Hooks

Short Description

Hooks for simplifying usage with GAMS.

Demos & Examples

useJsGlobal and useDatastream (Basic communication with GAMS)

  • useJsGlobal will check for an object with given property-name (data AND gui ) at the window object. Then exposes data as state.

  • useDatastream will request given datastream on given gams-object. (default: current object + "APP_DATA" datastream)

  • Try to prefer useJsGlobal over useDatastream because of complexity.

const GamsHooksDemo: React.FC = () => { // hook returns data and gui prop of global variable. // define type of returned property via ts const { data, gui }: {data: string, gui: string} = useJsGlobal("demo_conf"); // use datastream works the same (awaits global json at a gams-object) // empty will request current pid object const dsData = useDatastream(); // configure via params const dsData2 = useDatastream("context:fercan"); // requests default datastream APP_DATA const dsData3 = useDatastream("context:fercan", "MY_DATASTREAM_NAME"); // // example for requesting different datastreams depending on object. // assign datastream name as second param - do conditional logic to provide name. const example = useDatastream(null, window.location.href.includes("mode_main") ? "MAIN_DATASTREAM" : "SECONDARY_DATASTREAM"); // use returned values. return ( <div> {data ? <p>Data applied!</p> : null} <p>{data}</p> </div> ) }; export default GamsHooksDemo;

useSolr

  • requests data, exposes state and basic functionality for handling Apache SOLR in GAMS context.

  • requires the default configuration of Apache SOLR

  • First argument: URL of my exposed SOLR endpoint -> defaults to "http://gams.uni-graz.at/pardus/search/select"

  • Second argument: Predefine facets and initial facet fields.

// import of hook (could also import types from here) import { useSolr } from "zim-gamshooks"; const Demo: React.FC = () => { // call and use exposed properties. const { // method to toggle facet fields toggleFacetField, // (= "search-json") solr returned facet-fields and counts facetFields, // solr result // (= "result-json") result, // ref(!) facetQueryObject } = useSolr("https://my-solr-endpoint.org/search?", {"dc_type": [], "dct_medium": [], "edm_agent": []}); return <p>SOLR demo component</p> }
